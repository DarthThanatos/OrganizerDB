To update this database, you should:

1. Clone this repository using "git clone <url-address>" on your computer.
2. Open your command prompt and change directory to the destination where you have cloned OrganizerDB.
3. Being there, compile Main.java (if it is not compiled), using "javac -d . Main.java" command - but if the file organizer/Main.class exists, you can omit this step.
4. Being in OrganizerDB directory, run this manager' main class/method by typing "java organizer.Main"
5. Follow the instruction in the program
6. One you've added everything you wanted, push changes to this repository giving your commit a transparent name or at least attaching the version of that push (e.g. "DBManager i", where i is the index of change you've made)
